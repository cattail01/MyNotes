RT-Thread MicroPython

## 安装 MicroPython 依赖工具（命令行）

```bash
# 1. 串口/文件交互工具（Pymakr 依赖） 
pip install adafruit-ampy rshell 
# 2. 固件烧录工具（ESP32/ESP8266） 
pip install esptool
```
