## 引子

### AI的本质

* AI是根据概率生成下一个token
* 多个token组合生成一句话

### 提示词

* 是一种控制模型行为的方式

---

## 高质量prompt核心：丰富、具体、少歧义

> 把AI当人看，把它当个特别聪明但较真的实习生，把话说到位。

### 5个核心要素

#### 1. 角色定义

告诉AI它是谁，给它定义一个最匹配任务的角色。

> 角色决定了"视角"。你给AI定义角色，它就会调取那个角色的专业表达方式、思考框架和行业标准。

**示例**：

```
你是一名专业的跨境电商文案专家。
擅长撰写符合平台规范、转化率高的产品描述。
```

#### 2. 任务描述

告诉AI要做什么，越具体越好。

> 具体到你自己都觉得"这是不是太啰嗦了"的程度。

**示例**：

```
根据用户输入的产品信息：
1. 提取产品核心卖点
2. 生成3条不同风格的推广文案
3. 确保每条不超过50字
```

#### 3. 输入参数

告诉AI要知道什么，给出与任务相关的参数。

> **无关的参数，反而会稀释提示词，让AI抓不住重点。**

**示例**：

```
请根据以下参数生成回复：
- 产品名称：{product_name}
- 目标用户：{target_audience}
- 文案风格：{style}
```

#### 4. 输出格式

告诉AI以什么格式输出「如：格式要求、字数要求等」

**标准**：统一JSON输出、锁死顺序和数量、强制字段完整性、去AI化

**示例**：

```
严格以JSON格式输出，不得包含任何解释性文字：
{"title": "xxx", "content": "xxx", "tags": ["x", "x"]}
```

#### 5. 示例参考

> **这是最容易忽略但效果最好的要素。示例比任何描述都直观。**

**示例**：

```
输入：退货流程是什么？
输出：
{
  "type": "FAQ",
  "category": "售后",
  "answer": "请登录账号查看退货政策..."
}
```

### 限制条件（提升技巧）

AI有时候会跑偏，明确告诉AI不要什么，帮它划定边界。

> 明确说"不要什么"，本质是给AI划定边界，帮它排除无效选项，让生成结果更聚焦。

**示例**：

```
- 禁止编造产品参数
- 禁止添加Markdown格式
- 禁止超过指定字数
```

## 调优技巧

### 三步走法则

> **第一步**：写给自己看 —— 先用大白话写一遍，确认自己想清楚了
>
> **第二步**：写给AI看 —— 改成AI能理解的指令，加上角色、目标等
>
> **第三步**：迭代优化 —— 不满意就再问一遍，调整说法，加上例子，直到满意

### 迭代

找到好的提示词是个持续迭代的过程，多一个字或少一个字，对生成效果都可能有影响。

---

## 高手之路——提示词之外的能力

> 写好提示词只是工具。真正决定输出质量的，是写提示词背后的那个人。

* **审美能力**：见过足够多的爆款案例，心里有清晰的标准
* **判断能力**：哪些能用，哪些要改，哪些是胡说八道 —— 这来自你的专业知识和行业经验
