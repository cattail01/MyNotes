## 测试场景

| 测试用例 | 应用场景     | 具体需求描述                                                                                          |
| ---- | -------- | :---------------------------------------------------------------------------------------------- |
| 1    | 跨境电商智能客服 | 根据客户咨询的问题，自动从知识库中精准匹配相关信息，生成合规、简洁、礼貌的回复，不编造信息，不偏离知识库内容，直接可发送给买家。                                |
| 2    | 数据分析报告撰写 | 基于电商月度销售数据（包含销售额、客单价、复购率、热销品类）撰写分析报告，要求明确数据对比维度（同比、环比）、核心结论、优化建议，自动生成结构完整、数据清晰、有结论与优化建议的正式分析报告。 |
| 3    | 广告文案生成   | 为跨境电商产品撰写适配 Facebook、Google 投放的短广告文案，精准突出产品核心卖点与转化痛点，严格控制字符长度，语言简洁有吸引力、符合海外投放规范，支持多版本差异化输出。     |
